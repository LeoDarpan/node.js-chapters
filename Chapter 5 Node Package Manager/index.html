Chapter 5
Node.js Package Manager

NPM

  • NPM is the node package manager
  • It helps to install any package globally
  • To use npm we need to fire commands on the terminal as per the package we want to install
  • Just go the package download page and see the commands it says to fire using npm
  • To install the packages globally, -g is used in the command.
  • To install third-party packages and scripts, a package.json file is made to keep a track of them.
  • To make the file just type 'npm init' in the terminal and answer the prompts.
  • To insall the packages locally, go to the npm page, search for the package.
  • Use terminal to run the commands.
  • Older versions of node.js required a manual save into the dependencies file, but recent versions do it automatically.
  • When it comes to sharing code, the nod_modules folder is not shared.
  • This means, the packages are not shared. Therefore, to run the code successfully without errors of missing dependencies, all the packages must be installed again.
  • This is quite easy as, package.json file has all the dependencies listed and the 'npm insall' command installs all of them.
