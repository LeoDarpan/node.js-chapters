Node.js Client and Server

Clients and Server

  • Browsers sends requests to servers
  • IP addresses are uniqueand browsers always connect to IP addresses maintained by various DNS
  • Browsers requests HTTP from domains and gets back responses which are again in HTTP.
  • HTTP governs all the communication.
  • How to make Node a server.
    • The following code creates a server and makes it actively listen for requests.
      •                             
const http = require("http");

const server = http.createServer((req, res) => {
    console.log("Request made!");
});

server.listen(3000, 'localhost', () => {
    console.log("Listening for requests on port 3000!");
});
    • This marks the successful creation of a server. After running this code, if you go to the browser and try to access localhost:3000, you will see Request made! into the terminal.
