Node.js Intro

What is Node.js?

  • Node.js is a compiler that let's JS run outside of the browser
  • Written in C++, it runs on V8 engine, which is responsible for handling JS inside the browser.
  • It can
    • do read/write operations on local files,
    • connect to a database,
    • and also act as a server itself.
  • Node server runs JS on the backend and serves requests made by the browsers using that JS.
  • Benefits of learning node over other server side languages:
    • No extra language needed as Node compiles JS itself
    • it can share code on both the ends (front and end)
    • Huge community support
    • Large number of third party packages and tools to help
  • After installation command prompt terminal can be used as terminal to run node.js
  • All the JS instructions can be run in terminal after writing the command node and pressing enter
  • To change the directory in terminall type .. to navigate to the parent directory one by one till the super parent directory or drive
  • Use cd to change directory and write the path to the directory where the files are present.
  • Use mkdir to create a new directory.
  • Type Code . , to open VS Code from the directory of work using the terminal.
  • Just for knowledge
    • . - Means current directory
    • .. - Means parent directory
  • Script files can also be run from the terminal in the VS Code itself.
  • Just direct the terminal to the directory where the file rests and write the command: node filename(no need to write extension)
